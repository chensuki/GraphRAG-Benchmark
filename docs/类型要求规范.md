```python

    self,
    working_dir: str,
    llm_model_name: str,
    embedding_model_name: str,
```

| embedding_model_name | str | ✓ | 嵌入模型名称 |
| llm_base_url | Optional[str] | | API地址，可为None |
| llm_api_key | Optional[str] | | API密钥，可为None |
| config_path | Optional[str] | | 配置文件路径，可为None |
| top_k | int | | 检索数量，默认10 |
| chunk_size | int | | 文本块大小，默认256 |
| chunk_overlap | int | | 文本块重叠，默认32 |
| temperature | float | | 温度参数，默认0.0 |
| max_tokens | int | | 最大token数，默认2000 |

|--------|------|------|------|

## 返回数据类型要求

### 1. 查询结果（QueryResult）

推荐使用dataclass定义标准返回类型：

```python
from dataclasses import dataclass
from typing import List, Optional, Dict, Any

@dataclass
class QueryResult:
    """标准查询结果类型"""
    
    # 必需字段
    answer: str                      # 字符串，必需
    context: List[str]               # 字符串列表，必需
    
    # 可选字段
    entities: Optional[List[Dict[str, Any]]] = None
    relationships: Optional[List[Dict[str, Any]]] = None
    chunks: Optional[List[Dict[str, Any]]] = None
    passages: Optional[List[Dict[str, Any]]] = None
    sentences: Optional[List[Dict[str, Any]]] = None
    metadata: Optional[Dict[str, Any]] = None
```

**类型要求详细说明**：

#### answer字段
```python
# ✅ 正确
answer: str = "这是答案文本"

# ❌ 错误
answer: List[str] = ["答案"]      # 不能是列表
answer: Dict = {"text": "答案"}   # 不能是字典
answer: None                      # 不能是None（可以用空字符串）
```

**必须**：字符串类型，即使是错误消息也要是字符串

#### context字段
```python
# ✅ 正确
context: List[str] = ["片段1", "片段2", "片段3"]  # 字符串列表
context: List[str] = ["单个片段"]                # 单个片段也要在列表中
context: List[str] = []                          # 空列表表示无上下文

# ❌ 错误
context: str = "单个片段"             # 不能是字符串
context: List[Dict] = [{"content": "片段"}]  # 不能是字典列表
context: None                        # 不能是None（用空列表）
```

**必须**：字符串列表，即使只有一个片段也要放在列表中

### 2. 字典格式的返回（替代方案）

如果不使用QueryResult对象，可以返回字典：

```python
def query(self, question: str, **kwargs) -> Dict[str, Any]:
    """返回字典格式"""
    return {
        "answer": str,           # 必需，字符串
        "context": List[str],    # 必需，字符串列表
        "entities": Optional[List[Dict]],    # 可选
        "relationships": Optional[List[Dict]], # 可选
        "metadata": Optional[Dict]            # 可选
    }
```

**关键要求**：
- `answer` 必须是字符串
- `context` 必须是字符串列表

### 3. 仅返回答案（简化版本）

某些框架可能只返回答案字符串：

```python
def query(self, question: str, **kwargs) -> str:
    """仅返回答案字符串"""
    return "这是答案"
```

**注意**：这种方式需要额外的方法获取上下文：

```python
def get_context(self, question: str, **kwargs) -> List[str]:
    """获取上下文列表"""
    return ["片段1", "片段2"]
```

### 1. 检索评估（retrieval_eval.py）

评估脚本期望的数据格式：

```python
# 输入数据格式（predictions_*.json）
[
    {
        "id": str,                           # 必需，字符串
        "question": str,                     # 必需，字符串
        "context": List[str],               # 必需，字符串列表
        "evidence": str,                     # 必需，字符串
        "question_type": str                 # 必需，字符串

```

**评估脚本处理逻辑**：

```python
# 从评估脚本中可以看到：
context = item.get('context', [])
if isinstance(context, str):
    context_list = [context] if context else []
elif isinstance(context, list):
    context_list = context
```

**重要**：评估脚本会处理以下情况：

## 类型验证和转换

### 1. 参数验证函数

**推荐使用灵活的类型转换**（参考评估脚本的实际处理方式）：

```python
from typing import Optional, Any, List

def ensure_context_list(context: Any) -> List[str]:
    """
    灵活的context转换（与评估脚本一致）
    
    参考: Evaluation/retrieval_eval.py:270-277
    """
    if context is None:
        return []
    
    if isinstance(context, str):
        # 字符串转单元素列表
        return [context] if context else []
    
    if isinstance(context, list):
        # 确保列表中的每个元素都是字符串
        return [str(item) if not isinstance(item, str) else item for item in context]
    
    # 其他类型转换为字符串列表
    return [str(context)]

def ensure_string(value: Any, default: str = "") -> str:
    """确保返回字符串"""
    if value is None:
        return default
    if isinstance(value, str):
        return value
    return str(value)

def ensure_int(value: Any, default: int = 10, min_val: int = 1, max_val: int = 100) -> int:
    """确保返回有效整数"""
    if value is None:
        return default
    try:
        int_val = int(value)
        return max(min_val, min(max_val, int_val))
    except (TypeError, ValueError):
        return default
```

### 2. 在框架中使用验证

```python
class MyRAGFramework:
    def __init__(
        self,
        working_dir: str,
        llm_model_name: str,
        embedding_model_name: str,
        **kwargs
    ):
        # 灵活处理参数
        self.working_dir = ensure_string(working_dir, default="./workspace")
        self.llm_model_name = ensure_string(llm_model_name)
        self.embedding_model_name = ensure_string(embedding_model_name)
        self.top_k = ensure_int(kwargs.get('top_k'), default=10)
        
    async def aquery(self, question: str, top_k: Optional[int] = None, **kwargs) -> QueryResult:
        """查询方法，带灵活的类型处理"""
        # 处理输入参数
        question = ensure_string(question)
        top_k = ensure_int(top_k, default=self.top_k)
        
        # 执行查询...
        answer, context = await self._internal_query(question, top_k)
        
        # 灵活处理输出（不会抛出异常）
        context = ensure_context_list(context)
        answer = ensure_string(answer, default="无法生成答案")
        
        return QueryResult(
            answer=answer,
            context=context
        )
```

**关键原则**：
1. **不抛出异常**：类型转换失败时使用默认值，而不是抛出错误
2. **灵活处理**：接受多种输入类型，统一转换为期望格式
3. **参考实际实现**：与评估脚本的处理方式保持一致

### 3. 类型转换工具

```python
def ensure_string(value: Any) -> str:
    """确保返回字符串"""
    if value is None:
        return ""
    if isinstance(value, str):
        return value
    return str(value)

def ensure_string_list(value: Any) -> List[str]:
    """确保返回字符串列表"""
    if value is None:
        return []
    if isinstance(value, str):
        return [value] if value else []
    if isinstance(value, list):
        return [ensure_string(item) for item in value]
    return [str(value)]

def ensure_dict(value: Any) -> Dict[str, Any]:
    """确保返回字典"""
    if value is None:
        return {}
    if isinstance(value, dict):
        return value
    return {"value": value}
```

## 常见类型错误和解决方案

### 1. context字段类型错误

**错误示例**：
```python
# ❌ 错误：context是字符串而不是列表
result = {
    "answer": "答案",
    "context": "单个上下文片段"  # 错误！
}
```

**解决方案**：
```python
# ✅ 正确：context必须是列表
result = {
    "answer": "答案",
    "context": ["单个上下文片段"]  # 正确！
}

# 或使用验证函数
result = {
    "answer": "答案",
    "context": ensure_string_list(context_value)  # 自动转换
}
```

### 2. answer字段类型错误

**错误示例**：
```python
# ❌ 错误：answer是None
result = {
    "answer": None,  # 错误！
    "context": ["片段"]
}

# ❌ 错误：answer是字典
result = {
    "answer": {"text": "答案"},  # 错误！
    "context": ["片段"]
}
```

**解决方案**：
```python
# ✅ 正确：answer必须是字符串
result = {
    "answer": "答案文本",  # 正确！
    "context": ["片段"]
}

# 或处理None情况
answer = query_result.get("answer")
result = {
    "answer": answer if answer else "无法生成答案",  # 正确！
    "context": ["片段"]
}
```

### 3. question参数类型错误

**错误示例**：
```python
# ❌ 错误：question是列表
result = rag.query(["问题1", "问题2"])  # 错误！

# ❌ 错误：question是字典
result = rag.query({"text": "问题"})  # 错误！
```

**解决方案**：
```python
# ✅ 正确：question必须是字符串
result = rag.query("这是一个问题")  # 正确！

# 批量查询需要循环处理
questions = ["问题1", "问题2"]
results = []
for q in questions:
    result = rag.query(q)  # 正确！
    results.append(result)
```

### 4. top_k参数类型错误

**错误示例**：
```python
# ❌ 错误：top_k是字符串
result = rag.query("问题", top_k="10")  # 错误！

# ❌ 错误：top_k是浮点数
result = rag.query("问题", top_k=10.0)  # 错误！

# ❌ 错误：top_k是负数
result = rag.query("问题", top_k=-5)  # 错误！
```

**解决方案**：
```python
# ✅ 正确：top_k必须是整数
result = rag.query("问题", top_k=10)  # 正确！

# 或使用验证
top_k = int(top_k_value) if isinstance(top_k_value, (int, float)) else 10
top_k = max(1, min(100, top_k))  # 限制范围
result = rag.query("问题", top_k=top_k)  # 正确！
```

### 5. content参数类型错误（索引构建）

**错误示例**：
```python
# ❌ 错误：content是列表
rag.build_index(["段落1", "段落2"])  # 错误！

# ❌ 错误：content是字典
rag.build_index({"text": "内容"})  # 错误！
```

```python
# ✅ 正确：content必须是字符串
rag.build_index("完整的文本内容...")  # 正确！
```

## 总结

### 必须的类型要求

#### 传入参数
1. **字符串参数**：question、content、working_dir、model_name等
2. **整数参数**：top_k、chunk_size、chunk_overlap等
3. **可选参数**：使用Optional[T]类型提示

#### 返回数据
1. **answer字段**：必须是字符串（str）
2. **context字段**：必须是字符串列表（List[str]），使用单数形式 `context`
3. **所有字段**：必须存在，类型正确

### 核心原则

1. **类型一致性**：传入参数和返回数据的类型必须一致
2. **列表优先**：context字段始终使用列表格式，即使只有一个元素
3. **字符串安全**：所有字符串字段确保不为None
4. **灵活处理**：参考评估脚本的灵活类型转换方式
5. **错误安全**：类型错误时返回安全的默认值，不抛出异常

### 与评估脚本的兼容性

**关键点**：
- 评估脚本会自动处理 `context` 字段的类型转换
- 字符串会自动转为单元素列表，空值转为空列表
- 建议框架直接输出正确的列表格式，减少评估脚本的处理负担
